# pluttaneON

**3D-печатная светящаяся вывеска "pluttan"**


</div>

Большая DIY светящаяся вывеска с надписью "pluttan", предназначенная для 3D-печати. Каждая буква разбита на пронумерованные печатные сегменты, чтобы крупные символы помещались на стандартный стол FDM-принтера, плюс общая базовая деталь для сборки и подсветки на светодиодной ленте.

## ■ Возможности

- ❖ **Части букв** — STL-файлы для букв P, l, u, t, a, разбитые на сегменты (P1/P2, l1/l2, u1/u2/u3, t1/t2, a1/a2)
- ❖ **Общая база** — единая деталь `Body1.stl`, используемая во всей сборке
- ❖ **Пластины 3MF** — четыре `.3mf`-проекта (1–4) с расстановкой сегментов по готовым к нарезке пластинам
- ❖ **Готово к подсветке** — собранные буквы предназначены для свечения с помощью светодиодной ленты, которую предоставляет пользователь

## ■ Как это работает

```
1. Откройте один из четырёх файлов пластин .3mf (или отдельные .stl-детали) в вашем слайсере.
2. Напечатайте сегменты букв: непрозрачным филаментом — основу, полупрозрачным — лицевые части букв.
3. Соберите пронумерованные сегменты в полные буквы, используя общую деталь Body1.stl.
4. Закрепите светодиодную ленту (предоставляется пользователем) за собранными буквами и подключите питание для достижения эффекта свечения.
```

## ■ Использование

Откройте файлы `1.3mf`–`4.3mf` в вашем слайсере (Cura, PrusaSlicer и др.) для готовых пластин, либо импортируйте отдельные `.stl`-детали. Печатайте непрозрачным филаментом для основы и полупрозрачным материалом для лицевых частей букв, чтобы получить эффект свечения, затем разместите светодиодную ленту за собранными буквами.
